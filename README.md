# Talking-Duck
A very simple talking duck

# Install the script

To install the script directly, use the following command:

**Con `curl`:**

```sh
curl -o duck https://raw.githubusercontent.com/davide-dipierro/Talking-Duck/main/duck && chmod +x duck && sudo mv duck /bin
```

# Run the script

```sh
duck anything
```
